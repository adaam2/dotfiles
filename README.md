# Keybindings

## Tmux

The prefix key for tmux commands is ` (backtick)

### Splits / windows

- *Split vertically* = Backtick + dash
- *Split horizontally* = Backtick + backslash
- *Kill pane* = Backtick + x
- *Convert pane to window* = Backtick + !
- *Next window* = Backtick + n

## Vim

- *Leader key* = `,` (comma)
- *Escape key* = `jk`
- *Show file tree* = Ctrl + t
- *Search in project files* = Ctrl + f
- *Search by file name* = Ctrl + p
- *Toggle commenting of code* = Leader + c + space
- *Open current line on Github* = Leader + g + h + o
